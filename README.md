# 🪐 ФизМатИнфо (FizMatInfo)

Образовательный портал для школьников с лентой полезных материалов, разборами задач и системой персонализированных домашних заданий.
https://swordbird2002.github.io/fiz_mat_info_dv/index.html

![Status](https://img.shields.io/badge/status-active-success) ![License](https://img.shields.io/badge/license-MIT-blue)

## Особенности проекта

*   **Умная лента материалов:**
    *   Карточки с полезными материалами по Математике, Физике и Информатике.
    *   Фильтрация контента по предметам.
    *   Поддержка изображений, видео и формул.
    *   Эффектный 3D-тильт (наклон) карточек при наведении.
    

*   **Современный интерфейс:**
    *   Адаптивный дизайн (Mobile-first).
    *   Стеклянный морфизм (Glassmorphism) в элементах UI.
    *   Переключение тем (Светлая / Темная).

## Технологии

Проект написан на чистом стеке веб-технологий без использования тяжелых фреймворков:

*   **HTML5** — семантическая верстка.
*   **CSS3** — анимации, Flexbox/Grid, Bootstrap 5.
*   **JavaScript (ES6+)** — асинхронная загрузка данных (Fetch API), логика фильтрации и рендеринга.
*   **Vanilla-Tilt.js** — библиотека для 3D-эффектов на карточках.
*   **Bootstrap Icons** — иконки интерфейса.
*   **MathJax** — поддержка формул и уравнений LaTEX.
*   **Pyodide** — поддержка выполнения кода Python в браузере.
